Increment / Decrement Operator (==, --) : increment or decrement the value of variable by 1

++ Increment value of variables by 1

-- Decrement value of variables by 1

Postfix Operator
exp++; operators come after the variable
* first complete all other operators and then increment the value of variable


Prefix Operator
++exp;  operators come before the variable
* first increment the value of variable and then complete all other operations
